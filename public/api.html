Free API Usage

Rate Limit: Free API access is limited to 10 requests per day per IP address. For higher limits, please consider our Premium or Enterprise plans.

Random WiFi Name

GET /.netlify/functions/random

Returns a random funny WiFi name from our database.

Example Response:

{
  "name": "Pretty Fly for a WiFi"
}

Bulk WiFi Names

GET /.netlify/functions/bulk?count=5

Returns multiple WiFi names at once. Use the count parameter to specify how many names you want (max 100).

Example Response:

{
  "names": [
    "Pretty Fly for a WiFi",
    "The LAN Before Time",
    "Wi Fight the Inevitable",
    "Bill Wi the Science Fi",
    "The Promised LAN"
  ]
}

Category Filtering

GET /.netlify/functions/categories?name=movie

Returns WiFi names from a specific category.

Available Categories:

  • movies
  • puns
  • tech
  • warning
  • food
